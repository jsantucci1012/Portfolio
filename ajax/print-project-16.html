Bear Necessities' Coloring Book

Project Info:

For my Advanced Design class in college, a handful of students and myself decided to create a coloring book for the children in the Bear Necessities Pediatric Cancer Foundation.

We would each create individual themed pages and bring them together to create one book.

Project Details:

  • Company:Bear Necessities Pediatric Cancer Foundation
  • Tools Used:Adobe Illustrator
  • Date:Fall 2012
